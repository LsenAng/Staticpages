菜单栏功能介绍

Menu Bar Functions

1、添加蓝牙设备
1、 Add Bluetooth Device

在菜单中，点击添加蓝牙设备，则弹出蓝牙设备列表，选择则连接该蓝牙设备。

In the menu, click Add a Bluetooth device, a pop-up list of Bluetooth devices, select the Bluetooth device is connected.

2、确认地理位置
2、 Confirm location

在菜单中，点击确认地理位置，则获取当前的地理位置信息，并保存作为锁的位置信息，用于自动开锁功能。

In the menu, click OK location, obtain current location information and the location information is stored as a lock for the automatic lock function.

3、修改蓝牙PIN码
3、 Modify Bluetooth PIN

在菜单中，点击修改蓝牙PIn码，则跳转到设备信息界面，用于修改当前应用和锁蓝牙的PIN码信息。

In the menu, click Edit Bluetooth PIn code, jump to the device information interface for modifying the current application and the Bluetooth PIN code lock information.

4、校准自动开锁
4、Calibration automatically unlock

在菜单中，点击校准自动开锁，则弹出提示框，提示用户移动到距离锁正前方1.5米的位置，按下确定按钮，并获取一分钟的蓝牙RSSI信号，用于自动开锁时，校准是否允许开锁。

In the menu, click the calibration automatically lock, then a pop-up box that prompts the user to move from the front of the lock position 1.5 m, press the OK button, and get a minute of Bluetooth RSSI signal, for automatic unlocking, whether to allow calibration unlock.

5、恢复出厂设置
5、reset

在菜单中，点击恢复出厂设置，则弹出提示框，提示用户该怎样进行恢复出厂设置操作。

In the menu, click restore factory settings, pop-up boxes, prompts the user how to perform a factory data reset.

侧边栏功能介绍

Sidebar Features

1、蓝牙开锁
1、 Bluetooth unlock

在侧边栏中，点击蓝牙开锁，则进入蓝牙开锁界面，可以进行蓝牙开锁操作，并显示锁的电量和开锁次数。

In the sidebar, click Bluetooth unlock, then enter the unlock Bluetooth interface, Bluetooth unlock operation and displays the power lock and unlock times.

2、密码管理
2、 Password Management

在侧边栏中，点击密码管理，则进入密码管理界面，该界面显示锁中的所有的开门密码，可删除、编辑指定密码，也可以添加新的密码，最多添加两个密码。

In the sidebar, click the password management, then enter the password management interface that displays all the lock open password, you can delete, edit specify a password, you can also add a new password, add up to two passwords.

3、指纹管理
3、 Fingerprint Management

在侧边栏中，点击指纹管理，则进入指纹管理界面，该界面显示锁中的所有的指纹信息，可删除、编辑指定指纹，也可以添加新的指纹，最多添加255个指纹。

In the sidebar, click fingerprint management, then enter fingerprint management interface that displays all of the fingerprint lock information, delete, edit the assigned finger, you can also add a new fingerprint, add up to 255 fingerprints.

4、记录查询
4、 Records check

在侧边栏中，点击记录查询，则进入记录查询界面，该界面显示锁中的所有的开锁记录，可删除指定记录和按日期查询记录，记录按当前日期显示。

In the sidebar, click on the records check, then enter the record query interface that displays all the lock unlock records, you can delete the specified records and query records by date, record date according to the current display.

5、基本设置
5、 Basic settings

在侧边栏中，点击基本设置，则进入基本设置界面，该界面显示锁中的本机名称、电量、外围设备信息，可选择是否开启自动开锁功能，可进行手势密码设置，可输入配对PIN码删除外围设备。

In the sidebar, click General Settings, then enter the basic settings interface that displays the name of the native lock, power, peripheral device information, you can choose whether to open the automatic lock function, gesture password can be set, you can enter the pairing PIN code elimination peripherals.
